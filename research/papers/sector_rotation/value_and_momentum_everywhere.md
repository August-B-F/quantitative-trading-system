# Value and Momentum Everywhere

**HIGH PRIORITY**

- **Authors:** Clifford S. Asness, Tobias J. Moskowitz, Lasse Heje Pedersen
- **Year:** 2013
- **Source:** The Journal of Finance, Vol. 68(3), pp. 929-985
- **Links:** [AQR](https://www.aqr.com/Insights/Research/Journal-Article/Value-and-Momentum-Everywhere)

## Core Signal / Method

Documents consistent value and momentum premia across 8 diverse markets and asset classes. Value and momentum are negatively correlated with each other but positively correlated across asset classes, suggesting a common factor structure.

## Application to 8-ETF Rotation System

Combine momentum ranking with a value tilt. When selecting among 8 ETFs, blend momentum score with a valuation metric (e.g., P/E ratio for equity ETFs, yield spread for bond ETFs). The negative correlation between value and momentum provides diversification benefit.

## Known Failure Modes

- Momentum crashes (sudden reversals)
- Value traps (cheap assets that stay cheap)
- Combined strategy reduces but does not eliminate drawdown risk
