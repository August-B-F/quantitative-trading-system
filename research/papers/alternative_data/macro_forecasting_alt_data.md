# Macro Forecasting Using Alternative Data

- **Authors:** Apurv Jain
- **Year:** 2019
- **Source:** SSRN / ScienceDirect

## Core Method

Use web-scale "Big Data" to nowcast macroeconomic indicators that are reported with lag.

## Alternative Data Sources

- Web search trends (Google Trends)
- Social media sentiment
- Real-time consumer transaction data
- Geolocation data
- Mobile app usage
- Credit card payments
- Satellite imagery
- Weather data

## Key Finding

Alt data can address noise, lack of richness, and publication lag in traditional macro data. Web search data can predict non-farm payrolls.

## Application to 8-ETF Rotation System

Use Google Trends, credit card spending data, and sentiment indicators as leading regime indicators. Faster regime detection than waiting for official economic data.

## Known Failure Modes

- Short history makes backtesting unreliable
- Structural breaks in data
- Cost of alternative data
- Signal decay as more players use same data
